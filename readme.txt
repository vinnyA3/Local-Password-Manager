This is a simple, yet very useful password manager application that allows you to store you passwords for your web accounts.  The passwords are encrypted for your security.

Install:
  -Install node if you havent' already done so
  -Navigate to application directory
  -Run this command to install dependencies: npm install
  
Run and Use:  
  -To create an account run: node app.js create -n <AccountName (ie.Facebook)> -u<username/email> -p<password> -m<master password(used to encrypt and store your info- Save This!!>
  
  -To retrieve an account run: node app.js -n<AccountName> -m<master password>
  
~~~Enjoy!~~~
