This is a simple, yet very useful password manager application that allows you to store you passwords for you web accounts.  The passwords are encrypted for you security.
To run and use the app:
  -install node if you havent' already done so
  -To Create an account run: node app.js create -n <AccountName (ie.Facebook)> -u<username/email> -p<password> -m<master password(used to encrypt and store you info- Save This!!>
  -To retrieve an account run: node app.js -n<AccountName> -m<master password>
  
~~~Enjoy!~~~
